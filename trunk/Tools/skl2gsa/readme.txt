skl2gsa -
  Overview:
     This tool can be used to convert any number of skl files to a gsa file used by the skeleton code.

  Useage:
	Useage: perl skl2gsa.pl <base skl file> [<anim skl file> ... <anim skl file> ] >> output.gsa
	         <base skl file> - the base skeleton file w/out any animation (single frame)
                 <anim skl file> - the base skeleton animated e.g. Walk\n\n";

  You can test your gsa file in the test world.
